# groups

### `Used to` : display groups

> Ex: groups

- which all groups current user connected to

> Ex: cat etc/groups

- Shows all the groups

# groupadd

### `Used to` : add group

> Ex: groupadd [NAME]

- add group

# groupdel

### `Used to` : delete group

> Ex: groupdel [NAME]

- delete group

# Add user to a group

> Ex: gpasswd -a [USERNAME] [GROUPNAME]

# remove user to a group

> Ex: gpasswd -d [USERNAME] [GROUPNAME]
