# kill

### `Used to` : kill processes

### `Syntax` :- kill [OPTION]... PID

> Ex: kill 3286

- Kills process with PID 3286

> Option-1 : "-KILL"

- Ex: kill -KILL 3286

  If simple kill 3286 can't kill process somehow

> Option-2 : "-9" (sick kill)

- Ex: kill -9 3286

  If simple kill 3286 can't kill process somehow
